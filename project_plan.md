# 민호민아 성장앨범 프로젝트 계획서

## 📋 프로젝트 개요
- **프로젝트명**: 민호와 민아의 성장 앨범
- **도메인**: minhominah.com
- **기술 스택**: HTML/JavaScript + Supabase + Vercel
- **최종 결정**: HTML/JavaScript 기반으로 통합 완료 (2025-07-09)
- **호스팅**: Vercel (무료, CDN, 자동 배포)

## 🎯 기술 스택 (Vercel 최적화)

### Frontend
- **HTML5 + CSS3 + JavaScript**: 기본 웹 기술
- **Tailwind CSS**: 빠른 스타일링 (CDN 사용)
- **Supabase JavaScript SDK**: 백엔드 연동

### Backend
- **Supabase**: 
  - Database (PostgreSQL)
  - Authentication
  - Storage (이미지/비디오)
  - Real-time features

### 호스팅 환경
- **Vercel**: 
  - 무료 호스팅
  - 전 세계 CDN
  - 자동 HTTPS
  - Git 연동 자동 배포

